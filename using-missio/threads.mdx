<Info>
  **Overview** Threads in Missio.ai allow you to group related discussions and
  inquiries together, making it easier to track and manage ongoing
  conversations. Threads are added based on the source connections you have
  connected, such as Discord, Slack, and Email.
</Info>

## Creating Threads

### Source Connections

1. **Discord:** Threads can be either public threads from a Discord server, private threads from a Discord server, or Missio AI threads which automatically group messages into predicted threads using AI. Discord threads follow the Discord data schema.
2. **Slack:** Each message in a channel is treated as a new thread.
3. **Email:** Threads follow the standard email thread structure.

## Managing Threads

### Thread Metadata

Threads in Missio have thread metadata, where information such as labels, priority, status, assignment, etc., is stored.

### Edit a Thread

1. In the "Threads" section, locate the thread you want to edit.
2. Click on the "Edit" icon next to the thread name.
3. Update the thread name, description, or metadata (labels, priority, status, assignment, etc.) as needed.
4. Click "Save" to apply the changes.

### Delete a Thread

1. Find the thread you want to delete.
2. Click on the "Delete" icon next to the thread name.
3. Confirm the deletion in the prompt that appears.

## Using Threads

### Adding Messages to a Thread

1. When viewing a support inquiry or discussion, click on the "Threads" dropdown.
2. Select the appropriate thread from the list.
3. The message will be added to the thread, grouping it with related discussions.

### Viewing Messages in a Thread

1. In the "Threads" section, select the thread you want to view.
2. All messages and discussions related to that thread will be displayed, allowing you to see the full context of the conversation.

## Benefits of Using Threads

<Info>
  **Organization** Threads help in grouping related discussions, making it
  easier to track ongoing conversations and context.
</Info>

<Info>

  of discussions, reducing the chances of missing important information.

<Info>

  ensuring that your team can easily follow and manage conversations.
